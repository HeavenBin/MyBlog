    空格(‘ ‘)

    1空格 = N空格/1换行 建议：1空格
    N空格 =

    换行(‘ ‘+’回车’)

    1换行 = 2空格+1换行 建议：2空格+1换行
    2换行 = 2换行 建议：2换行
    N换行 =

    *斜体 或者 _斜体_ 斜体*，_斜体_

    **粗体 或者 __粗体__ 粗体**，粗体
